# Инструкция по установке обновлений

## 1. Установка новых зависимостей

Проект использует Poetry для управления зависимостями. Установите новую библиотеку для перевода:

```bash
poetry add deep-translator
```

Или, если используете pip:

```bash
pip install deep-translator
```

## 2. Применение миграции базы данных

Была создана миграция для добавления поля `name_rus` в таблицу `players`.

### Для применения миграции на сервере:

```bash
# Если используется alembic напрямую
alembic upgrade head

# Или через python
python -m alembic upgrade head
```

### Для отката миграции (если нужно):

```bash
alembic downgrade -1
```

## 3. Новые функции

### В админ-панели (http://31.222.229.78/admin/utils):

1. **Перевод имен всех игроков** - автоматически переводит имена всех игроков на русский и сохраняет в поле `name_rus`

2. **Перевод имени одного игрока** - переводит имя конкретного игрока по его ID

### В админ-панели игроков (http://31.222.229.78/admin/player):

- Добавлено поле `ID` для ручного ввода при создании игрока
- Добавлено поле `name_rus` для отображения и редактирования русских имен

## 4. Технические детали

- Используется библиотека `deep-translator` с Google Translate API
- Перевод происходит автоматически с любого языка на русский
- Все изменения логируются в консоль приложения
